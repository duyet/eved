# Research

When a question is novel, ambiguous, or needs current/external facts, gather evidence
first, then answer.

1. Choose the source:
   - About Duyet, his blog, CV, projects, or GitHub → the **duyet** connection.
   - Anything else on the web → the **firecrawl** connection: `firecrawl_search` to find
     pages, `firecrawl_scrape` to read one, `firecrawl_map` / `firecrawl_crawl` for a
     whole site, `firecrawl_extract` for structured data.
2. Cross-check anything important against a second source.
3. Answer concisely: conclusion first, then the key facts, the source links, and whatever
   is still uncertain.
